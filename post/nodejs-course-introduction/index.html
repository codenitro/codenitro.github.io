Node.js, Express and MongoDB

An introductory course building API's using Node.js, Express and MongoDB

 Posted on February 17, 2021  |  2 minutes  |  307 words  |  Aseem Gaurav

NodeJS Course - Basics

Node.js is an open-source, cross-platform, back-end JavaScript runtime environment that runs on the Chrome V8 engine and executes JavaScript code outside a web browser. Node.js lets developers use JavaScript to write command line tools and for server-side scripting—running scripts server-side to produce dynamic web page content before the page is sent to the user’s web browser. Wikipedia

NodeJS is widely used in building api’s and server backend for web and mobile apps. If you know javascript, its very easy to get started with NodeJS. In this course we will introduce you to node.js, how it works, the event loop. We will also cover how to use node to run javascript code, we will also introduce you to creating a server using express and mongodb.

This course is for you if:

  1. You want to learn and get started with Node.js backend development
  2. You want to learn how to create backend API’s for web or mobile apps
  3. You want to know how to use MongoDB in a Node.js API
